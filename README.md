# 微信公众号排版器 (WeChat Article Style Generator)

✨ **一个现代化的微信公众号和多平台图文排版工具，基于 Vue 3 + Vite 构建。**

本项目旨在帮助内容创作者轻松制作高颜值、高质感的卡片式排版。内置多达数十种酷炫且高级的纯代码排版风格，实现“即选即用、一键分发”的极速体验。支持纯 CSS 绚丽特效如：浩瀚星河、丁达尔光束、极光粒子、高级毛玻璃等。

## 🌟 主要功能

- 🎨 **43 种精美排版效果库**：内置庞大的渲染引擎，分类包含渐变、毛玻璃、极客、经典等。
- 📋 **一键复制 (微信编辑器完美适配)**：所有样式完全遵循微信公众号编辑面板支持的内联 CSS 规则编写。全选复制后直接粘贴到微信公众平台后台，**不掉格式、不丢边距**。
- 🖼️ **高清卡片导出**：基于 `html2canvas` 深度优化，所有的排版卡片均支持在浏览器内直接“一键下载为 PNG 图片”。智能裁切了白底边框并保留圆角，适合快速发布至小红书、知乎、微博等依赖配图的平台。
- ☁️ **公众号草稿箱直连**：内置一套基于 Express 的后端轻量代理，配置 `AppID` 和 `AppSecret` 后即可实现排版全选一键同步到你的微信公众号草稿箱。
- 📝 **长文自动排版分割**：输入长文，系统会自动按段落将长文本无感式切分为统一格式的多组件排版序列，极大地解放双手。

## 🚀 快速启动

### 方案 1：仅运行前端（用于浏览器排版与图片导出）

如果你只需要在浏览器里进行排版、复制内容格式或导出为图片，仅启动前端项目即可。

```bash
# 1. 安装依赖
npm install

# 2. 启动前端 Vite 服务
npm run dev
```
打开 `http://localhost:5173/` 即可直接体验。

### 方案 2：全栈运行（需要配置微信公众号一键草稿同步）

如果你需要调用微信 `draft/add` API，请同时运行前端和 Node.js 后端以避免客户端跨域拦截并保护密钥安全。

```bash
# 1. 启动前端
npm run dev

# 2. 在另一个终端进入后端目录并安装依赖
cd server
npm install

# 3. 启动后端代理服务
node index.js
```
进入页面左上方的「微信同步」，填入相应配置即可开始直传。

## 🛠 技术栈
* 前端基础层: Vue 3 (Composition API), Vite, 纯 CSS Token 系统
* 工具包: html2canvas (高清节点截图), Vue Reactivity API
* 后端代理: Node.js, Express, axios, cors
